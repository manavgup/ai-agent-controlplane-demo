Do it all yourself

Run the entire governed mesh on your own machine. Build the tool with Bob, then govern it end to end.

Everything runs on your laptop — the gateway, the policy sidecar, six MCP servers, two cross-language A2A agents, and Bob. The gateway is at localhost:4444. One command builds the finished mesh; or walk the four stages and watch Bob build a tool and then govern it.
Heavy on conference WiFi (~10 Docker images). On a slow network, the Codespaces path is gentler.

1Clone & build the whole mesh

One command brings up the finished, governed stack and proves it.

git clone https://github.com/manavgup/ai-agent-controlplane-demo cd ai-agent-controlplane-demo make quickstart
You'll see the full mesh come up and finish 16 passed, 0 failed. Want to drive Bob too? Install it: curl -fsSL https://bob.ibm.com/download/bobshell.sh | bash. The proof runs without Bob — Bob only drives the demo.

2Or walk the four stages yourself

Each make command runs on your laptop; each 🗣 prompt runs in your Bob (it auto-connects — make bob / bob-operator / bob-install-builder write the connection for you, no make connect locally).

① Build — Bob writes the server, ungoverned

Create a new MCP server with fastmcp at mcp-servers/sales-tax/server.py: a tool add_tax(amount, rate_pct=8.5) returning {amount, rate_pct, tax, total}, plus a GET /health route, served over HTTP on 0.0.0.0:8000.
make stage1-build
You'll see add_tax(100, 8.5) → tax=8.50, total=108.50 on bare :8000 — no token, no policy, no audit. That exposure is the problem the next stage fixes.
If Bob's file won't run: make stage1-scaffold drops in a known-good server, then re-run make stage1-build. Don't keep asking Bob to regenerate over a working file.

② Govern — register → grant → call

make stage2-govern
Operator  Register the sales-tax service with finbyte-gateway at http://sales-tax:8000/mcp.
make salestax-grant
Builder  Add sales tax to $100.
The throughline registering only catalogs the tool — granting is what makes it callable. The tool you ran wide-open on :8000 now returns 108.50 governed, token-gated through one seam.

④ Verify — the proof

make verify-controls
You'll see 16 passed, 0 failed — the full governed mesh proven. Same end-state as make quickstart, but you watched it get built.

3Inspect it

Both commands auto-open your browser at localhost — no port-forwarding.

make monitor # Admin UI → localhost:4444/admin
make inspect-mcp # MCP Inspector → localhost:6274
make inspect-a2a # A2A Inspector → localhost:8090
Reset between runs: make stage-reset (stops the bare server + container, deletes the generated server.py) or make demo-reset for a clean slate.
