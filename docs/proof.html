How to run it yourself

git clone <repo>
cp .env.example .env
make up            # gateway + OPA + 4 MCP servers + 2 A2A agents (Python + Rust)
make seed          # register everything + FinOps/Treasury virtual servers
make verify-controls   # 16/16 — proves all four controls + cross-language A2A
make companion     # browser companion on http://localhost:7070
make bob-config    # emit .bob/mcp.json for IBM Bob

Gateway: ContextForge v1.0.2. Controls enforced at the gateway tool hooks (tool_pre_invoke / tool_post_invoke), including on bridged A2A agent calls. See docs/ARCHITECTURE.md and docs/RUNBOOK.md.
